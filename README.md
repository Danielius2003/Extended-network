# Extended-network
An extended network for 3 buildings
## Physical, logical topologies and documentation
### Logical topology includes:
- Ipv4 addressing
- Ipv6 addressing
- working PAT for Ipv6
- web server
- DNS server for Ipv6
- VLANs
- ACLs
### Physical topology icludes:
- Everything that is in Logical topology
- Shows how everything is seperated
### Documentation for:
- Ports
- VLANs
- Cables used
